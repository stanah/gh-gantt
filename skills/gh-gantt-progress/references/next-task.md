# 次タスク提案フロー

オープンタスクから次に着手すべきタスクを提案する。

## プロセス

### 1. 着手可能なタスクの絞り込み

```bash
gh-gantt list --state open --unblocked --sort end_date
```

ブロッカーが解消済みで期限が近い順にタスクを表示する。

### 2. 候補タスクの詳細確認

上位の候補について `gh-gantt show <id>` で body・blocked_by・priority を確認する。

### 3. 提案の優先度

以下の観点で優先順位を付けて提案する:

1. **期限が近い** — end_date が直近のタスク
2. **ブロッカーが解消された** — blocked_by が全て closed になったタスク
3. **優先度が高い** — priority が設定されているタスク
4. **他タスクのブロッカーになっている** — クリティカルパス上のタスク

### 4. 結果提示

候補を一覧で提示し、ユーザーに選択を促す。
タスクが選択され着手する場合は `gh-gantt-workflow` に引き継ぐ（Status 更新やブランチ作成は workflow が担当）。
